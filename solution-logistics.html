How it works

물류의 모든 단위를 제품 단위로 연결합니다

포장 계층을 수학적 구조로 묶어, 상위 한 번의 스캔으로 하위 전체의 무결성을 검증합니다.

01 · Hierarchical Grouping

단품부터 팔레트까지 하나의 트리로

단품(C)–박스(P)–케이스–팔레트(GP)–LOT을 Parent-Child로 연결합니다. 어느 단위에서 스캔하든 상·하위를 즉시 조회해 정밀 역추적의 기반이 됩니다.

  • 계층 코드 자동 매핑 (GP·P·C·LOT)
  • 어느 단위 스캔에서도 상·하위 조회
  • 단품까지 정밀 역추적
24박스 내 단품 EA

* 제품 예시 화면 · 교체 예정

02 · Blind Check

박스를 열지 않고 전수검수합니다

상위 포장 코드를 1회 스캔하면 내부 구성·수량·누락을 즉시 검증합니다. 박스를 개봉하지 않고 전수검수가 끝나 물류 검수 시간을 획기적으로 단축합니다.

  • 미개봉 무결성 검증 · 누락 즉시 탐지
  • WMS 입출고 검수 자동화
  • 검수 소요 대폭 단축
블라인드 체크 결과 화면제품 스크린샷 영역 (교체 예정)
03 · Targeted Recall

문제 범위만 정확히 회수합니다

이상이 발생한 LOT·범위만 정밀하게 좁혀 회수합니다. 전량 회수 대비 비용과 브랜드 리스크를 크게 줄이고, 영향 범위를 데이터로 소명합니다.

  • 영향 LOT·범위 자동 특정
  • 회수 대상 QR 동적 라우팅(리콜 안내)
  • 회수 이력 감사 원장 기록
표적 리콜 범위 화면제품 스크린샷 영역 (교체 예정)
04 · Return Verification

반품을 단순 서류가 아닌 데이터로 검증합니다

입고 반품의 단품 시리얼을 스캔해 자사 정식 출고분인지, 중복·이상 이력은 없는지 즉시 확인합니다. 가품을 되파는 악의적 반품을 접수 단계에서 차단합니다.

  • 정식 출고 여부·중복 스캔 즉시 판정
  • 가품 역유통 반품 사전 차단
  • 부당 CS 접수 단계 필터링
