How it works

복제할 수 없는 코드, 설치 장벽 없는 인증

규칙 없는 1회성 난수 코드와 서버 대조로, 위·변조·중복 스캔을 즉시 식별합니다.

01 · Random Serial

제품마다 다른 256-bit 난수 시리얼

모든 제품에 규칙 없는 개별 난수 키를 부여합니다. 다음 값을 추정·복제할 수 없어 표면이 같은 QR이라도 내부는 모두 다릅니다.

  • 시리얼당 256-bit 난수 키
  • 복제·추정 원천 차단
  • 활성화 전(DP) 코드는 미인증 처리
256bit 난수 키

* 제품 예시 화면 · 교체 예정

02 · No App

전용 앱 없이, 카메라로 즉시 판정

소비자는 휴대폰 기본 카메라·웹으로 스캔하면 서버 대조로 즉시 정품 여부를 확인합니다. 설치 장벽 없이 인증률을 끌어올립니다.

  • 기본 카메라·웹 스캔 · 앱 불필요
  • NFC 대비 칩 비용 0
  • 다국어 인증 안내
모바일 스캔 인증 UX제품 스크린샷 영역 (교체 예정)
03 · Anomaly Detection

위·변조와 중복 스캔을 즉시 식별

동일 코드가 반복 인증되거나 물리적으로 불가능한 거리를 이동하면 위조·복제로 분류합니다. 미등록 코드·중복 스캔을 즉시 차단하고 브랜드에 알림합니다.

  • 중복·재스캔 이상 탐지
  • 비현실적 이동거리 탐지
  • 미등록 코드 즉시 차단·알림
정상 스캔 1,240

* 제품 예시 화면 · 교체 예정

04 · Brand Experience

인증이 곧 브랜드 경험이 됩니다

정품 확인과 동시에 이벤트·사용법·재구매 등 브랜드 콘텐츠로 연결합니다. 물리(봉인 라벨)와 디지털(난수+서버 검증)을 결합한 2중 보안으로 신뢰를 완성합니다.

  • 인증 직후 이벤트·재구매 유도
  • 봉인 라벨 + 난수QR 2중 보안
  • 블록체인 앵커링으로 위·변조 무력화
